# Personalized Books Platform Prototype

This repository contains a minimal Node.js Express prototype for an AI-based platform generating personalized children's books. It demonstrates a few placeholder API endpoints.

## Setup

Install dependencies and start the server:

```bash
npm install
npm start
```

The API will listen on port `3000` by default.

## Available Endpoints

- `POST /api/register` – dummy user registration
- `POST /api/login` – dummy login
- `POST /api/generate-story` – generates a placeholder story structure

These endpoints should be extended with real authentication and AI integration.
